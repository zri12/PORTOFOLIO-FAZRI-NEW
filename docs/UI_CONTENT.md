Comments should feel human, short, and moderated. Do not generate fake testimonials as proof of business success. Admin replies should be concise and clearly separate from visitor content.

## Certificates

Certificate copy should include title, issuer, category, issue date, credential ID, credential URL, and image when available. Do not invent credential authority or verification status.

## Admin Content

Admin labels should be practical and consistent. Avoid playful language in admin CRUD screens. Public copy fields should use the same approved terminology used in this document.
